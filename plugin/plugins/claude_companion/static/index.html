🐾 Claude 伙伴陪伴

当你在终端里和 Claude 一起奋斗时
你的 NEKO 伙伴会在这里默默关注你、温暖你

🌸 使用教程

从零开始，手把手带你接入 Claude 陪伴功能

🎀 第一步：了解这个插件能做什么
这个插件的作用很简单：当你在终端里用 Claude Code 写代码时，它会自动记录你在做什么（修改了哪个文件、调试了什么问题、提交了什么代码），然后把这些信息「悄悄告诉」你的 NEKO 猫娘。猫娘就会在对话框里关心你、鼓励你，就像有一个小伙伴在旁边陪你写代码一样温暖 ☁️
🔧 第二步：配置 Claude Code 的 hooks（核心步骤）
找到以下目录中的 settings.json 文件：

C:\Users\你的用户名\.claude

具体操作：右键点击 settings.json → 选择「在记事本中编辑」

然后在文件里完整插入以下内容（通过复制粘贴的方式）：
{ "hooks": { "Stop": [ { "matcher": "", "hooks": [ { "type": "http", "url": "http://localhost:48920/hook/turn-end", "timeout": 10 } ] } ] } }
⚠️ 注意：如果你的 settings.json 里已经有其他内容，请确保将上述 "hooks" 部分合并进去，注意 JSON 的逗号语法。如果不确定，可以先把整个文件内容截图发给 AI 助手帮你合并。
💾 第三步：保存文件
编辑完成后，在记事本左上角的 「文件」 菜单里点击 「保存」 选项即可。

保存后可以关闭记事本。
✅ 第四步：确认插件已启用
打开 NEKO 主界面，在插件列表中找到 「Claude 伙伴陪伴」，确认它处于启用状态。

当插件运行时，下方活动时间线会开始记录你的编码活动。你可以在当前页面的状态栏中看到连接状态。
🚀 第五步：开始体验！
打开终端，运行 Claude Code 开始写代码。

当你在终端里操作时，NEKO 猫娘会自动收到你的活动通知并在对话框里回应你～ 你可以在下方设置面板里修改用户名和推送冷却时间。

🎉 现在你可以放心写代码了，你的伙伴会一直在这里陪着你！
💛 温馨提示： 插件会在你每次完成一轮对话后自动推送，两次推送之间至少有 60 秒 的冷却时间（可在下方设置面板调整）。 如果暂时不想被打扰，可以通过设置面板调大冷却时间。
